# Weil Germany Microsite - Requirements Analysis Dashboard

Interactive dashboard for the de.weil.com project requirements analysis.

## Access

- **URL**: https://www.discover.kamanin.at/weil/
- Protected by login/password

## Development

Edit `weil/index.html`, commit, and push. GitHub Pages auto-deploys from `main` branch.

## Stack

Single HTML file with React 18 + Babel standalone (CDN). No build step required.
